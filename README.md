# Proyecto Flask con OpenStreetMap

Aplicación web desarrollada con Flask que permite buscar ubicaciones y mostrarlas en un mapa interactivo usando OpenStreetMap y Leaflet.

## Requisitos
- Python 3
- Flask
- Requests
